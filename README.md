# 🍅 Pomodoro Timer

A simple and modern Pomodoro Timer application built to improve focus and productivity using the Pomodoro Technique.

## ✨ Features

- ⏱ Start, pause, and reset timer
- 🍅 Pomodoro work sessions
- ☕ Short and long break sessions
- 🔔 Sound notification when session ends
- 📱 Responsive and clean UI
- 🌙 Easy-to-use minimalist design
